Outils de Cybersécurité

MENU_SHELL

Administration

Script Bash interactif regroupant des outils essentiels pour l'administration système et réseau en environnement Linux. Simplifiez vos tâches quotidiennes avec une interface en ligne de commande intuitive.

Fonctionnalités :

  • Visualisation et gestion des interfaces réseau
  • Recherche de fichiers et gestion des processus
  • Analyse réseau (scan de ports, détection d'appareils)
  • Gestion des règles de pare-feu iptables
  • Administration des imprimantes
Bash Networking System Admin Security

SecLogParser

Analyse

Outil d'analyse de logs de sécurité permettant de détecter des comportements suspects et de générer des alertes personnalisées.

Fonctionnalités :

  • Analyse de logs Apache, Nginx, Windows Event
  • Détection d'attaques par pattern matching
  • Corrélation d'événements multi-sources
  • Alertes configurables (mail, Slack, Discord)
Python Regex ElasticSearch

VulnTracker

Veille

Outil de suivi et d'alerte sur les nouvelles vulnérabilités publiées, avec possibilité de filtrage par critères.

Fonctionnalités :

  • Agrégation de données depuis NVD, CVE, ExploitDB
  • Filtrage par produit, vendeur, score CVSS
  • Détection d'exploits disponibles
  • Alertes par email ou webhook
